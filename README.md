Cybersecurity Home Lab
Overview

This repository documents the home lab I built to learn Linux administration, Docker, monitoring, authentication, and security operations through hands-on experience.

Rather than following tutorials without understanding them, every service has been installed manually, integrated with the rest of the environment, documented, and tested.

Current technologies include:

Ubuntu
Docker
Docker Compose
Nginx
Homepage Dashboard
Uptime Kuma
Prometheus
Grafana
Wazuh SIEM
Authentik SSO
Tailscale

The project is continually expanded as I learn new technologies.

Goals
Learn Linux administration
Learn Docker
Build monitoring infrastructure
Learn SIEM operation
Learn identity management
Learn infrastructure documentation
Simulate enterprise environments
